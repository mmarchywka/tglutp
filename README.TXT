
----------------------------------------------------------------------------
UI SUMMARY
----------------------------------------------------------------------------
Abbreviations used:
   LMB - left mouse button
   MMB - middle mouse button
   RMB - right mouse button

Selection:
   LMB click          : selects a picking ray

Camera Navigation:
   Alt+LMB drag       : orbit
   Alt+MMB drag       : pan
   Alt+LMB+MMB drag   : dolly
   Ctrl+LMB drag      : pitch & yaw
   Ctrl+MMB drag      : zoom
   Ctrl+LMB+MMB drag  : roll
   'l'                : look at centre of the cube
   'r'                : reset camera


   If you have a 2-button mouse, use
   Alt+LMB drag       : orbit
   Shift+LMB drag     : pan
   Shift+Alt+LMB drag : dolly


Other Options:
   'd'                : cycle display mode (wireframe and flat/smooth shaded)
   RMB                : menu of options, showing hotkeys
